Este APP é um projeto utilizando Angular + Node ou Json - É um cadastro de pessoa

Base de dados:
1 - A base de dados deve ter duas tabelas: pessoa e profissão

API:
1 - A API deve ter um cadastro para pessoa e profissão
2 - Uma Pessoa só pode ter uma profissão
3 - A API deve ter um endpoint para pessoa e outro para profissão
4 - Para ambos você tem que construir operações CRUD

Frontend:
1 - Deve ter uma página para listar todas as pessoas
2 - Ao clicar no botão "ler mais" deve redirecionar para "detais da pessoa" página mostrando apenas a pessoa selecionada
3 - Deve permitir cadastrar nova pessoa
4 - Deve permitir editar a pessoa
5 - Deve permitir remover pessoa
6 - Ao clicar no botão remover deve aparecer a tela modal perguntando se você tiver certeza sobre a exclusão, se sim, remova se não, não faça nada
